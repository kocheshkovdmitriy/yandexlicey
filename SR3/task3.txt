Напишите программу для систематизации прошлого.

Через аргументы командной строки передаются хост и порт сервера и дополнительные данные в порядке:

хост;
порт;
произвольное количество комнат.
Также могут быть переданы дополнительные опции командной строки:

--future – все не кратные значению аргумента числа дополнить до ближайшего большего кратного значению аргумента числа;
значение по умолчанию 2;
--prev – если аргумент передан, то все кратные значению аргумента числа разделить нацело на значение аргумента.
Каждое число списка проверяется так: сначала по первому условию, а если оно не выполнено, то по второму, если оно
передано, и именно в таком порядке. Если первое условие выполнено, по второму не проверяется.

На этом сервере находится словарь с информацией о количестве вещей в каждой комнате в разные моменты времени. Получив с
сервера все данные, преобразуйте числа для указанных в аргументах командной строки комнат в соответствии с правилами,
описанными в дополнительных опциях.

Затем нужно получить список словарей с ключами:

room – комната;
things – общее количество вещей в ней;
bigger – количество раз, когда следующее число было не меньше предыдущего (первое число сравнивать не с чем, поэтому
оно не считается).
Полученный список, упорядоченный по названию комнаты, запишите в файл past.json.

Примечание
Для проверки вашего решения на локальной машине:

Запустите сервер в PyCharm или консоли, он будет активен пока не введете слово "stop"
Введите номер примера (1 или 2)
Запустите своё решение и проанализруйте результат
Cкрипт для запуска сервера: past.py

# Пример запуска:
python3 solution.py 127.0.0.1 5000 master_bedroom bath --future 3
